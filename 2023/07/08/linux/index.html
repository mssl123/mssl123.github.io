gcc

gcc -o 输出文件 待编译文件

过程:预处理,编译,汇编,链接

  1. 先将各个.c文件预处理为.s文件
  2. 再将.s汇编为.o
  3. 最后将各个.o链接为可执行文件

每次执行gcc时都会将整个过程完整执行一遍

Makefile

目标:依赖
命令

  1. 通配符(patten):

    1. $@ 目标
    2. $< 第一个依赖
    3. $^ 所有依赖

  2. 假想目标
    .PHONY:目标

  3. make命令
    make 目标(如不输入目标,则默认第一个目标)

  4. 变量

    1. 即刻变量 a:=值 #a的值即刻确定
    2. 延时变量 b=值 #b的值再使用到时才确定

  5. 关于头文件

    1. gcc -M .c #打印.c的依赖
    2. gcc -M -MF .d .c #把依赖写入.d文件
    3. gcc -c -o .o .c -M -MF .d #编译.o,并将依赖写入.d

文件目录

bin

binary的缩写,主要存放一些常用命令

boot

启动的核心文件

dev

device,设备文件

etc

存放用户所需要的配置文件和子目录

home

用户

lib

library,主要存放一些动态库供应用程序调用

lost+found

一般为空,系统非法关机后,相关文件会存放于此

media

自动挂载一些linux系统自动识别的设备,如U盘,光驱等

mnt

一般为空,提供给用户用于(手动)挂载临时的文件系统,比如另外的硬盘等

opt

提供主机额外安装软件所需的目录

proc

虚拟的目录,系统内存的映射,可以通过直接访问这个目录来获取系统信息,与进程相关

root

超级用户的目录

sbin

super user bin

srv

主要存放一些系统服务启动后所要用的数据

run

主要存放一些系统运行时需要用到的文件

usr

主要用于存放用户应用程序及文件,类似windows的program files
包括bin,sbin,src(其中bin,sbin,srv就是上述提到的,只不过上述三个类似于快捷方式)

tmp

存放临时文件

var

主要存放一些经常被修改的文件,比如日志文件,电子邮箱等

命令

终端文字颜色

  1. 白色:普通文件(文本,图片)
  2. 蓝色:文件夹
  3. 绿色:可执行文件(.sh)
  4. 青色:链接文件(快捷方式)
  5. 红色:压缩包
  6. 黄色:设备文件
  7. 灰色:其他
  8. 红色但有灰色背景:已经失效的链接文件

终端常用快捷键

文件权限

ls -al命令

  1. r:可读
  2. w:可写
  3. x:可执行/切换目录
  4. -:无权限
    例如:drwxrwxrwx
    d表示文件夹(-表示文件),后面9个,3个一组依次表示所有者(owner,创建文件的人)、用户组(group)、其他人(除所有者和用户组的人)的权限

通配符

  1. *:匹配一个或多个字符
  2. ?:匹配一个字符
  3. []:匹配方括号内任意一个字符
  4. [!]:除方括号内都能匹配到

常用命令

  1. man
  2. info:是man的详细信息
  3. whatis:等同于man -f(查询命令的执行结果)
  4. touch:改变文件时间戳属性
    1. 一个文件有三个时间属性
      1. access 创建时间
      2. modify 修改时间
      3. change
    2. 可以通过stat命令查询文件时间属性
    3. touch可以一次性创建多个文件
    4. 参数
      1. touch -a:只改变accesstime
      2. touch -m:只改变modifytime
      3. touch -c:不创建新文件,只改变时间戳
      4. touch -r:使用参照文件的时间戳
      5. touch -d:改为其他时间,而不是现在时刻的时间戳
      6. touch -t:改为其他具体时间戳
  5. mkdir:创建目录
    1. 参数:
      1. -p:递归创建多级目录
      2. -m:创建目录的同时,设置目录权限,(例如:mkdir -m 700 dir)
      3. -v:显示创建目录过程
  6. rm:删除文件或者目录

    1. 无法复原

    2. 参数

      1. -r:递归删除文件(即删除文件夹)
      2. -f:忽略不存在的文件,强制删除,于-r一起使用,-rf
      3. -i:删除时会询问用户是否删除
      4. -v:显示删除过程
  7. rmdir：删除空目录
    1. 参数
      1. -p：递归删除空目录
      2. -v：显示指令详细执行结果
  8. mv：移动/重命名文件
    1. 语法：
      1. mv 文件名(目录名) 路径
      2. mv 旧文件名(目录名) 新文件名(目录名)
    2. 参数:
      1. -i:若存在重名文件,询问用户是否覆盖
      2. -f:强制覆盖,不询问
      3. -b:若文件已经存在,则为其创建一个副本
      4. -u:只有当被移动对象的时间戳比目标路径中的同名文件新时,才能移动成功,并覆盖
    3. 结合通配符使用,例如:mv * ..
  9. cp:复制文件或目录
    1. 语法:
      1. cp 被复制文件名 新文件名
    2. 参数
      1. -f:直接复制,若文件名重复,则直接覆盖
      2. -i:重复时询问
      3. -b:创建备份后覆盖
      4. -r:复制目录
      5. -a:包括文件(读写权限等)属性一起复制,ll -a
  10. stat:文件详细信息
    1. 内容
      1. access time:访问时间(读取和修改时间)
      2. modify time:修改时间
      3. change time:状态改变时间(修改,文件属性变化)
    2. 参数
      1. -f:查看文件所在文件系统的信息
      2. -t:简洁方式输出
  11. rename：以字符串方式批量改变文件名
  12. basename:提取路径中的文件名
  13. dirname:提取路径中非目录部分
  14. chattr/lsattr:修改/显示文件权限
    1. 参数:
      1. -R:递归处理目录下的所有文件
      2. -v:设置文件或目录版本
      3. -V:显示命令执行过程
      4. +:开启文件或目录的该项属性
      5. -:关闭该项属性
      6. =:指定文件或目录的该项属性
    2. 属性(这里的属性与可读可写权限不同):
  15. file:识别文件类型/查看链接
  16. md5sum:生成和校验文件的md5值
  17. find:搜索文件
    1. 参数:
      1. find 路径 -name 所需查找文件的部分名字(通配符表示)
  18. which:搜索在path变量中的系统命令所在的路径
    echo $path
  19. whereis:查找命令的二进制程序路径,源码文件和man手册路径
  20. locate:快速查找文件或目录(在相关数据库查找,但数据库不是实时更新的,通过sudo updatedb更新)
  21. chown:改变文件或目录的用户和用户组
    1. 语法:
      1. sudo chown 用户:用户组 文件名
      2. sudo chown 用户 文件名
      3. sudo chown :用户名 文件名
      4. sudo chown -R 用户:用户组 目录名
  22. chgrp:改变用户组
  23. chmode:改变文件或用户权限
    1. 语法:
      1. chmod [ugoa][[+-=][rwx]] 文件名
      2. r->4
      3. w->2
      4. x->1
      5. u->user
      6. g->grup
      7. o->other
      8. a->all
    2. 例子:chmod a+r 文件名(所有用户都增加可读权限)
  24. grep:文本搜索
    1. 语法:grep 关键词 路径
    2. 参数:
      1. -h: 不显示文件名称
      2. -r:递归搜索
      3. -l:只列出含关键词的文件名
      4. -i:忽略大小写
      5. -c:统计出现次数
      6. -n:显示行号
    3. 搭配其他命令使用
      1. 命令 | grep “”
  25. uptime:查看系统负载
    1. 参数
    2. -p显示机器正常运行时间
    3. -s系统启动时刻
  26. free:显示系统内存使用情况
    1. 参数:
      1. -m:以MB为单位显示
      2. -k
      3. -g
      4. -h:智能选择单位
      5. -s:实时显示
    2. 使用:free -hs 3(每三秒刷新显示一次)
  27. iostat:监视系统输入输出设备和CPU使用情况
  28. scp:远程文件加密传输
    1. 语法:
      1. 将本地文件传到服务器
        1. scp 本地文件路径 服务器用户名@服务器地址 服务器文件存放路径
        2. scp -r
      2. 将服务器文件拷贝到本地
        1. scp 服务器用户名@服务器地址 服务器文件路径 本地文件存放路径
        2. scp -r
